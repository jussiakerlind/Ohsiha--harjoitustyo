# Ohsiha--harjoitustyo
Harjoitustyössäni tein sovelluksen, joka hakee käyttäjän sijainnin ip-osoitteen avulla ja näyttää säätiedot viidessäkymmenessä lähimmässä paikassa. Sovellus kertoo käyttäjän sijainnin sen hetkisen lämpötilan ja sään kuvauksen. Visualisoinnilla näytetään myös 50 lähimmän paikan lämpötila sekä ilmankostaus ja erilaiset säätilat kuvauksineen. Harjoitustyössä on myös toteutettuna tehtävänannon vuoksi käyttäjän luominen, sisään ja ulos kirjautuminen, sekä käyttäjän poistaminen.

Web-ympäristönä käytin djangoa ja datan visualisoinnit tein Highchartsilla
